<h1 align="center">Web-сервис мониторинга и анализа информации для закупочной и договорной деятельности АЭС</h1>

## Описание проекта
Проект представляет собой веб-сервис для мониторинга и анализа информации из открытых источников (OSINT) с применением методов веб-скрапинга. Основная цель сервиса — поддержка процессов закупочной и договорной деятельности атомных электростанций (АЭС) путем автоматизированного сбора, структурирования и аналитической обработки релевантной информации.

Сервис обеспечивает:  
- Автоматический сбор данных с публичных источников (новостные порталы, государственные реестры, тендерные площадки).  
- Нормализацию и хранение данных в структурированном виде.  
- Аналитическую обработку, включая поиск ключевых событий, трендов и рисков.  
- Уведомления о критических изменениях или новых данных.  

## Основные возможности
- **Веб-скрапинг и парсинг**: гибкое извлечение данных из разных источников с поддержкой изменений структуры страниц.  
- **Обработка и нормализация данных**: очистка, структурирование, агрегация.  
- **Аналитика и визуализация**: дашборды, отчеты, графики и KPI для закупочной и договорной деятельности.  
- **Интеграция с внутренними системами**: REST API для передачи данных внешним сервисам.  
- **Безопасность и конфиденциальность**: защита от несанкционированного доступа, соблюдение требований информационной безопасности АЭС.  

## Архитектура
Проект построен по микросервисной архитектуре:  
1. **Data Collector** – модуль веб-скрапинга и интеграции с API источников.  
2. **Data Processing** – очистка, нормализация и хранение данных (PostgreSQL / MongoDB).  
3. **Analytics Engine** – аналитическая обработка и генерация отчетов.  
4. **Web Interface** – пользовательский интерфейс с дашбордами (React / Angular).  
5. **API Layer** – REST API для интеграции с внешними и внутренними системами.  

## Технологический стек
- Backend: Python (FastAPI), Node.js  
- Frontend: React, TypeScript, Tailwind CSS  
- Базы данных: PostgreSQL, MongoDB  
- Очистка данных: Pandas, NumPy, BeautifulSoup, Scrapy  
- Очереди задач: RabbitMQ / Celery  
- Визуализация: Plotly, Grafana  
- Контейнеризация: Docker, Kubernetes  
- CI/CD: GitHub Actions, GitLab CI/CD
